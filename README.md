# NameForge — AI Business Name Generator

**16 naming strategies, brand kits, and AI-generated taglines.** Find the perfect name for your business, product, or startup.

🔗 **Live:** [https://atalovesyou.github.io/nameforge/](https://atalovesyou.github.io/nameforge/)

## Features

- **16 Naming Strategies** — Compound, portmanteau, abstract, domain-style, metaphor, action, and more
- **100 Prefixes + 100 Suffixes** — Creative word combinations
- **50 Abstract Roots** — Unique invented names
- **Brand Cards** — Color palettes, brand voice, target audience for each name
- **Favorites & History** — Save and revisit best names
- **PDF Brand Reports** — Downloadable brand kit with name, colors, voice, tagline
- **100% Client-Side** — No data leaves your browser
